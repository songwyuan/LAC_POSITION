# LAC_POSITION

## Step 0：安装 Python 环境和依赖
```
确认 Python 版本：

python3 --version
# 输出应为 Python 3.7.x
```

## 创建虚拟环境（推荐）：
```
python3 -m venv pbertkla_env
source pbertkla_env/bin/activate
```
